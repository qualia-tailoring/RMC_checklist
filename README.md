# 中小企業診断士 1次試験 学習チェックリスト

中小企業診断士1次試験の全7科目・全トピックを網羅した、ブラウザで動く学習管理ツールです。  
インストール不要。HTMLファイルをダウンロードするか、GitHub Pagesにアクセスするだけで使えます。
---

## 機能

### チェックリスト
- 7科目・全トピックを科目別に管理
- チェックした日時を自動記録
- 進捗率をリアルタイム表示（全体 / 科目別 / セクション別）
- データはブラウザのlocalStorageに保存。ページを閉じても消えません

### 忘却曲線スケジューリング
エビングハウスの忘却曲線に基づき、チェックしたトピックの復習タイミングを自動計算します。

| 復習タイミング | チェックからの経過日数 |
|---|---|
| 1回目 | 1日後 |
| 2回目 | 3日後 |
| 3回目 | 7日後 |
| 4回目 | 14日後 |
| 5回目 | 30日後 |

期限が近づくとトピック横にバッジが表示され、「復習スケジュール」タブで一覧確認できます。

### フラッシュカード
チェック済みのトピックをランダムに出題します。
- 「覚えた」→ 次のカードへ
- 「もう一度」→ 復習タイマーをリセット
- フィルターで「復習期限あり」や特定科目に絞り込み可能

### AI解説機能（オプション）
Anthropic APIキーを設定するとフラッシュカードでAIによる解説を生成できます。  
APIキーはご自身のブラウザにのみ保存され、サーバーには送信されません。

## 使い方

### GitHub Pagesで使う（推奨）
上部のリンクをブラウザで開くだけです。

### ローカルで使う
1. このリポジトリの `index.html` をダウンロード
2. ファイルをブラウザで開く

### AI機能を使う（任意）
1. サイドバー下部の「AI機能」をONにする
2. [Anthropic Console](https://console.anthropic.com/) でAPIキーを取得
3. キーを入力して「保存する」をクリック

---

## データについて

- すべてのデータはご自身のブラウザ（localStorage）にのみ保存されます
- 他のユーザーのデータとは完全に分離されています
- サーバーへのデータ送信は一切ありません（AI機能使用時はAnthropicのAPIに直接リクエストが送られます）
